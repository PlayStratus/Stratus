# Documentations

> Contains documentation for Stratus.

## Rendered on [playstratus.io](https://playstratus.io/#Blogs)

For the your documentation to be rendered on the website, add your `.md` file to this directory. The file must start with the yaml:

```yaml
---
title: <Title of your documentation>
author: <Your name>
image: <Path to image to be shown on the website, relative to the docs directory>
tags: [<Tags to categorize your documentation separated by commas>]
---
<Content of your documentation in markdown format>
```
