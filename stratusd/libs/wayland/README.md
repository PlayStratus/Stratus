# stratusd libwayland

This directory contains definitions of stable Wayland protocols from the
[`wayland-protocols`][wayland-protocols] repository as well as patched source
code from the [`wayland`][wayland] library. All changes are documented by
comments starting with "`STRATUS`". Unused source code has been commented out to
reduce the scope of changes, and may be removed in the future.

[wayland]: https://gitlab.freedesktop.org/wayland/wayland
[wayland-protocols]: https://gitlab.freedesktop.org/wayland/wayland-protocols
